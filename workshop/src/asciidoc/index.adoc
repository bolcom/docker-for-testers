= Docker for testers

**The purpose of this training is to give you a starting point in learning how to use Docker on your local machine. Getting to know Docker and the possibilities it opens up should give you ample material to start applying Docker in your day to day work. The training is structured as such, parts theory, parts practice, with every step building on the previous step.**
